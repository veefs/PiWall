# PiWall

![alt text](PiWall.png)

## What is PiWall?

Piwall is an opensource project that uses a raspberry pi 4b in order to act as a fireawll. It includes things like a web dashboard, network filtering, and more.

## Why use PiWall?

PiWall is spefically used to be more efficent than your regular PfSense configuration. In addition to this it's much easier to set up and requires a lot less technical skill.
